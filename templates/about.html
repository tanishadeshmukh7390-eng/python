About Us Page

Study Quiz Hub Portal is an educational web application developed using Python Flask, HTML, CSS, Bootstrap, and Jinja Templates. The project provides programming quizzes, theory notes, automatic result generation, percentage calculation, and pass/fail status. It helps students improve their programming knowledge and prepare for interviews and examinations through an interactive learning environment..
