# Study Quiz Hub

A modern Flask-based learning platform with a premium dark theme UI.

## Features

✨ **Modern UI**
- Premium gradient night theme with electric blue and teal accents
- Responsive design for all devices
- Smooth animations and glow effects
- Professional dark mode

📚 **Quiz Categories**
- Technology (Web Development, AI, Data Science, Cloud Computing, Cyber Security, Mobile App Dev)
- Programming Languages (C, C++, Java, Python)
- Computer Science (OS, DBMS, Networks, Data Structures)

🎯 **Functionality**
- User authentication (Login/Register)
- Student management
- Quiz interface with score tracking
- Admin dashboard for question management

## Installation

```bash
# Create virtual environment
python -m venv myenv
myenv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run application
python app1.py
```

Visit `http://localhost:5000` in your browser.

## Technology Stack

- **Backend**: Flask 3.1.3
- **Frontend**: Bootstrap 5.3.0 + Custom CSS3
- **Database**: SQLite
- **Language**: Python 3.10+
