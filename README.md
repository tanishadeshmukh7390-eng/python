# Study Quiz Hub

A Flask-based quiz and learning platform with a modern dark UI.

## Features

* Dark theme with electric blue and teal colors
* User login and registration
* Student management system
* Multiple quiz categories and subjects
* Score tracking and results

## Technologies Used

* Python
* Flask
* Bootstrap
* SQLite

## Run the Project

```bash
python -m venv myenv
myenv\Scripts\activate
pip install -r requirements.txt
python app1.py
```

Open `http://localhost:5000` in your browser.

## Project Structure

```text
app1.py
templates/
static/
requirements.txt
```
